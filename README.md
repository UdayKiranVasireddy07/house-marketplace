# 🏠 House Marketplace

A modern, responsive web application for browsing, listing, and managing houses.  
Built with **React**, **Supabase**, **Nominatim (OpenStreetMap)**, and **Leaflet**, styled for a recruiter-ready dark theme with gradient accents.

---

## ✨ Features
- 🔐 **Authentication**
  - Supabase Email/Password login
  - Google Login (OAuth integration)
- 🏡 **Property Listings**
  - Add, edit, and delete listings
  - Price tags with ₹ symbol
  - Timestamped entries
- 📍 **Location & Maps**
  - Integrated **Nominatim API** for geocoding (convert address → exact coordinates)
  - Interactive maps powered by **Leaflet**
  - Map markers for each property
  - Zoom, pan, and responsive map display
- 📱 **Responsive Design**
  - Optimized for mobile, tablet, and desktop
- 🎨 **Light Theme UI**
  - Gradient-based CSS
  - Subtle animations for recruiter-ready polish
- ⚡ **Fast Deployment**
  - Hosted on Vercel/Netlify
- 🔍 **Search & Filter**
  - Search by property name, location, or price

---

## 🛠️ Tech Stack
- **Frontend:** React, Context API, Reducers
- **Backend:** Supabase (Auth + Database)
- **Authentication:** Supabase Auth + Google OAuth
- **Maps & Location:** Nominatim (OpenStreetMap), Leaflet
- **Styling:** CSS3, Gradient-based themes, Responsive layouts
- **Deployment:** Vercel / Netlify
